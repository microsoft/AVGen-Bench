# Support

## How to get help

Use GitHub Issues for bug reports, feature requests, and usage questions related to AVGen-Bench.

Before opening a new issue:

- Search existing issues to avoid duplicates.
- Check [README.md](README.md) for setup steps, environment requirements, and module-specific commands.
- Confirm the problem is in this repository rather than in an upstream dependency under `third_party/` or a referenced evaluation submodule.

When filing an issue, include:

- The evaluation module you were running, for example `eval/Syncformer` or `eval/music_check`.
- The exact command you ran.
- Your environment details, including OS, CUDA version, Conda environment name, and relevant package versions.
- Full error logs and, if applicable, a minimal input example that reproduces the problem.

## Support scope

This repository is maintained on a best-effort basis. Maintainers may not be able to provide:

- Formal SLA-backed support.
- Custom environment debugging for all third-party dependencies.
- General support for upstream projects vendored or referenced in this repository.

For questions or issues in external dependencies, consult the corresponding upstream project when appropriate.

## Security issues

Do not report security vulnerabilities through public GitHub issues. Follow the reporting guidance in [SECURITY.md](SECURITY.md).

## Microsoft support policy

Support for this project is limited to the resources listed above.
